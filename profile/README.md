# MUST Custody

Institutional crypto custody platform — secure wallet management, policy controls, and tenant operations.

> Private organization. Sign in with your MUST account to access repositories.

## Platform

| Repository | Description |
| --- | --- |
| [**must-custody-api**](https://github.com/must-custody/must-custody-api) | API gateway — JWT / HMAC auth, request forwarding to Core |
| [**must-custody-core**](https://github.com/must-custody/must-custody-core) | Core service — business logic, Postgres, Celery, RabbitMQ |
| [**must-custody-admin**](https://github.com/must-custody/must-custody-admin) | Admin & tenant SPA (Vite + React) |
